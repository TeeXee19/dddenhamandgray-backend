
## Description
D&G Backend repository.

## Installation

```bash
$ pnpm install
```

## Seeding the Database
```bash
# Make migrations
$ pnpx prisma migrate dev --name <migration_name>
```

```bash
$ pnpm run admin
```
```bash
$ pnpm run states
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev
# OR
$ nest start --watch

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
